This is real time chat web app which works on websocket for real time chatting and mongodb altas for user data mangament. Thanks to websocket users can see online status of their percipients whom they are chatting with. Users can also chat to new user by their mobile numbers. It's basically same as whatsapp we use in out daily life.


Features :
1. Real Time chat using websocket
2. Implemented online users status
3. Clean and smooth UI
4. Get toasted notification

Backend Details:
1. Used JWT Tokken based auth
2. Used Webstocket for real time chat and online status
3. Build on React + Vite
4. Used React UI library for chat section and toasted notifications
